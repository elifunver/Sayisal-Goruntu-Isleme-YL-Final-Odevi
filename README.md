# Cilt Kanseri Görüntü İşleme Projesi

Bu proje, ISIC veri setindeki deri lezyonlarını analiz etmek için geliştirilmiştir.

## Kullanılan Teknikler
* **Segmentasyon:** Otsu Eşikleme
* **Morfoloji:** Closing ve CCL (En Büyük Alan Seçimi)
* **Öznitelik Çıkarımı:** İstatistiksel, Şekilsel ve Dokusal (GLCM)

##  Sonuç
2239 görüntü işlenmiş ve başarılı bir şekilde özellik tablosu oluşturulmuştur.
